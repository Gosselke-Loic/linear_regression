1. Install virtual venv 
    - virtualenv .venv

2. Select interpreter (in vscode)
    - Ctrl + Shift + p -> python: interpreter

4. How to activate a virtual env
    - source .venv/bin/activate

3. Install libraries
    - pip install numpy pandas

5. It might be necessary to restart vscode.

- How deactivate virtual env
    - deactivate

- Setup flake8
    - alias norminette=flake8
    - use: norminette
